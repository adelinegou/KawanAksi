# KawanAksi
Kawan Aksi is a dedicated web platform designed to connect volunteers with various available social activities hosted by local communities and NGOs.

#  Description
KawanAksi bridges the gap between people who want to make a social impact and organizations that need volunteer support. Through this platform, users can discover, view details, and register for social activities in their area — all in one place.

#  Key Features
🔍 Browse Activities — Explore a variety of available social events  
📋 Event Details — View complete information: location, date, time, organizer, and volunteer requirements  
📝 Volunteer Registration — Sign up as a volunteer directly through the platform  
📞 Contact Organizer — Reach out to event organizers with ease  

#  Tech Stack
HTML5 for Page Structure  
CSS3 for Styling and Layout  
JavaScript (E56) Module for Logic and Interactivity
